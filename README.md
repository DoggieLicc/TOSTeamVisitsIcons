# TOS Team Visit Icons

This mod adds visual icons to help you coordinate where your teammates are going to visit that night

## Features

### Role Revival Icon

Allows you to enable the ability to show the icon of roles been revived by your retri or necromancer, for example if your Necromance revives a Werewolf to visit someone, you will see a Werewolf icon on that person.

### Day Ability Icons

Allows you to enable the ability to show the icons of your teammates' Day abilites, for example if you have a Jailor on your team, there will be an icon on who they select to be Jailed furing the day.

Does not add icons for instant abilities (revealing, meteor, etc.)

### Display Mode

Allows you to switch how the icons from your teammates' regular abilites will be shown, the options are:

* **Role Icon**: You will see the role icon of your teammates, with exceptions made for roles that can target multiple people (Seer, Witch, Necromancer, etc.)
* **Ability Icon**: The icon of the ability your teammate is using will be shown
* **No Icon**: No icon will be added for your teammates action. Necronomicon and special abilities still appear as normal

### Book Icon

Allows you to switch how the Necronomicon icon appears.

* **Replace Icon**: Will replace the ability/role icon with the Necronomicon
* **Add Icon**: Will add the Necronomicon icon next to the default ability/role icon
* **No Icon**: Never show the Necronomicon icon

### Special Ability Icon

Allows you to switch how your teammates' special abilities appear.

* **Add Icon**: Will add the special ability icon next to the default ability/role icon
* **Replace Icon**: Will replace the ability/role icon with the special ability icon
* **No Icon**: Never show special ability icons

### Show Own Actions

Allows you to enable the ability to add icons for your own actions

* **Never**: Never add icons for your own actions
* **Only as Factional Evil**: Only add icons when you play as a factional evil (Coven, Apocalypse, Vampire, Cursed Souls)
* **Always**: Always add icons for your own actions

### Handle Overcharges

Allows you to enable the ability for the mod to try handle overcharged teammates properly.

* **Only Myself**: Only handle when you're overcharged
* **All Teammates (EXPERIMENTAL)**: Handle all teammate's overcharges. Experimental, as there's no 100% way to tell if someone else is actually overcharged
* **Never**: Never handle overcharges
